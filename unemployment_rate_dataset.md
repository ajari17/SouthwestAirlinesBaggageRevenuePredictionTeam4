# BTS Monthly Metric — Spreadsheet Summary

## Source
- Data provided by **Bureau of Labor Statistics (BTS)**.

- Downloaded from BTS and filtered to the relevant rows/columns.

## Purpose
- Present the cleaned, human-readable Markdown version of the monthly metric table (Year × Month).

- Values are numeric (floats).

## Pre-processing / Notes

- 2025 contains values only through August in the source.
